Performance Messages Sender

How to Use

  1. Ensure you are logged in to WhatsApp from this browser window.
  2. Download the template files provided for Driver and Performance data.
  3. Fill in the template files with the required data. Make sure the data is in the same format as the template.
  4. Upload the filled template files using the form below.
  5. Click on "Upload" to send the performance messages to drivers.

Upload CSV Files

Please upload two CSV files: one for Driver data and another for Performance data.
